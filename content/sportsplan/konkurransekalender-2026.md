## 6. Anbefalinger per aldersgruppe

### 6.1 Turnerings-progresjon

| Aldersgruppe | Anbefalt deltakelse | Antall turneringer/sesong |
|---|---|---|
| **13–15 år** | Interne + 2–3 kretstrunder | 7–8 |
| **16–17 år** | Interne + kretsserie + 1–2 regionale | 10–12 |
| **18–19 år** | Alle interne + kretsserie + regionale + evt. nasjonale | 12–15+ |
| **Academy** | Interne + klubbmesterskap + evt. kretsserie | 4–6 |

### 6.2 Kvalifiseringsvurdering

Før påmelding til ekstern turnering vurderer trener:

| Kriterium | Krav |
|---|---|
| IUP-kategori | Minimum G for krets, E for regional, C for nasjonal |
| HCP | Oppfyller turneringens HCP-krav |
| Turneringserfaring | Har gjennomført minst 2 interne turneringer |
| Mental beredskap | Trener vurderer at spilleren er klar (se [Mental trening](./mental-trening.md)) |
| Foresatte-samtykke | For juniorer under 18 |

## 7. Turneringsforberedelse og -evaluering

### 7.1 Forberedelse (1 uke før)

| Dag | Aktivitet |
|---|---|
| **7 dager før** | Meld på / bekreft deltakelse |
| **5 dager før** | Kursmanagement-gjennomgang (hvis kjent bane) |
| **3 dager før** | Mental forberedelse: mål, strategi, visualisering |
| **1 dag før** | Lett trening (kort spill + putting). Pakke bag. Tidlig leggetid |
| **Turneringsdag** | Oppvarmingsrutine (se [Kursmanagement](./kursmanagement-gfgk.md) §8) |

### 7.2 Evaluering (1–2 dager etter)

| Steg | Innhold |
|---|---|
| **Selvvurdering** | Spilleren fyller ut turneringsjournal (se [Mental trening](./mental-trening.md)) |
| **Statistikk** | Score, putts/hull, GIR, fairway-%, up-and-down |
| **Mental evaluering** | Fokus, selvsnakk, stresshåndtering, prestasjonsrutine (1–5) |
| **Samtale med trener** | 15 min — hva gikk bra, hva jobber vi med neste gang |
| **IUP-oppdatering** | Resultater og observasjoner registreres i IUP |
