# AK Golf Group Website

Website for AK Golf Group — premium golf development platform featuring AI-powered training plans, coaching services, and e-commerce.

## 🚀 Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Runtime:** React 19
- **Language:** TypeScript 5 (Strict mode)
- **Styling:** Tailwind CSS 4
- **Database:** Supabase (PostgreSQL + Auth)
- **Payments:** Stripe (Checkout + Subscriptions)
- **AI:** Anthropic Claude (Training plan generation)
- **Email:** Resend
- **PDF:** @react-pdf/renderer
- **Deployment:** Vercel

## 📁 Project Structure

```
├── app/                    # Next.js App Router
│   ├── api/               # API routes (Stripe webhooks, AI generation, contact)
│   ├── auth/              # Authentication (login, callback)
│   ├── coaching/          # Coaching landing page
│   ├── junior/            # Junior academy page
│   ├── produkter/         # Products (training plans, brand guide)
│   ├── treningsplan/      # Training plan flow (preview, dashboard)
│   └── layout.tsx         # Root layout with SEO metadata
├── components/
│   ├── website/           # Marketing site components
│   └── dashboard/         # Dashboard UI components
├── lib/
│   ├── ai/                # AI training plan generation
│   ├── cache.ts           # Caching utilities
│   ├── env.ts             # Environment validation
│   ├── pdf/               # PDF generation
│   ├── stripe/            # Stripe configuration
│   └── supabase/          # Database clients
├── hooks/                 # React hooks
├── content/               # Markdown content
└── docs/plans/            # Implementation plans
```

## 🛠️ Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Run linter
npm run lint
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🔐 Environment Variables

Create `.env.local` with the following variables:

```bash
# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key

# Anthropic AI
ANTHROPIC_API_KEY=sk-ant-api03-...

# Stripe
STRIPE_SECRET_KEY=sk_test_...
STRIPE_WEBHOOK_SECRET=whsec_...
STRIPE_PRICE_BASIS=price_...
STRIPE_PRICE_STANDARD=price_...
STRIPE_PRICE_PREMIUM=price_...

# Resend (optional)
RESEND_API_KEY=re_...

# App config
NEXT_PUBLIC_SITE_URL=https://akgolf.no
PORTAL_URL=https://portal.akgolf.no
CONTACT_EMAIL=post@akgolf.no
```

All environment variables are validated at startup using Zod (see `lib/env.ts`).

## 🏗️ Key Features

### AI Training Plans
- Personalized 12-week training plans generated by Claude AI
- Category-based progression system
- PDF download with branded design
- Preview mode with paywall

### Dashboard
- Subscription management (Standard/Premium)
- Weekly schedule view
- Exercise library
- Progress tracking
- Test protocols

### E-commerce
- Stripe Checkout integration
- One-time payments (Basis tier)
- Subscriptions (Standard/Premium)
- Webhook handling for payment events

### Security
- Content Security Policy headers
- Strict-Transport-Security
- X-Frame-Options, X-Content-Type-Options
- CSRF protection via Supabase Auth

### Performance
- Server-side caching with `unstable_cache`
- React `cache()` for deduplication
- Optimized font loading with `next/font`
- Image optimization

## 📝 Database Schema

Main tables in Supabase:

- `plans` — AI-generated training plans
- `profiles` — User profiles
- `subscriptions` — Stripe subscription records

Row Level Security (RLS) policies protect user data.

## 🧪 Testing Stripe Webhooks Locally

```bash
# Install Stripe CLI
stripe login

# Forward webhooks to localhost
stripe listen --forward-to localhost:3000/api/treningsplan/webhook
```
